# Glossary

Key terms used in MassGen.

---

**Agent**
An AI entity that performs tasks and collaborates with other agents.

**Backend**
Integration layer between MassGen and model providers (OpenAI, Claude, etc.).

**Case Study**
Documented real-world usage example showing how MassGen solved a problem.

**Consensus**
Process where multiple agents agree on a solution or approach.

**Dev Branch**
Active development branch for next release (e.g., `dev/v0.1.8`).

**MCP (Model Context Protocol)**
Protocol for integrating external tools and data sources with AI models.

**Orchestrator**
Core system that manages and coordinates multiple agents.

**Pre-commit Hooks**
Automated checks that run before allowing a commit (formatting, linting, etc.).

**PR (Pull Request)**
GitHub feature for proposing changes to the codebase.

**Track**
One of four contribution areas: Research, Case Studies, Marketing, or Development.

**Track Owner**
Person leading and coordinating a contribution track.
